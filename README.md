# chalkling
**Authors:** Jan Garong, Yining Wang, Ruo Ning (Nancy) Qiu

**Server Setup:**
1. Run ``mvn clean install`` to generate a jar file.
2. Run ``heroku local web`` to run the jar file on the web locally on your computer.
